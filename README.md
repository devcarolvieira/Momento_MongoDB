# 📊 Momento - MongoDB

## Sobre

Este repositório contém a resolução da atividade **Momento**, focada na utilização do MongoDB para consultas, agregações, atualizações e análises de dados corporativos.

A atividade aborda cenários práticos envolvendo funcionários, departamentos, escritórios, suprimentos e vendas, permitindo aplicar conceitos fundamentais e avançados de bancos de dados NoSQL.

## Tecnologias Utilizadas

- MongoDB
- MongoDB Compass
- Aggregation Framework

## Conteúdo

- Consultas básicas (`find`)
- Inserção, atualização e remoção de documentos
- Agregações com `Aggregation Pipeline`
- Relacionamentos utilizando `$lookup`
- Análises de dados e relatórios
- Consultas de auditoria e validação

## Objetivo

Praticar operações de manipulação e análise de dados no MongoDB, explorando desde consultas simples até agregações complexas e cenários próximos ao ambiente corporativo.
